1. System Architecture

System Information Commands

# Basic System Information
uname -a                  # Display all system information
cat /etc/os-release      # Distribution information
lscpu                    # CPU information
free -h                  # Memory information with human-readable sizes
df -h                    # Disk space usage
lsblk                    # Block devices information

# Process Management
ps aux                   # List all processes
top                      # Real-time process viewer
htop                     # Enhanced process viewer

2. Package Management

APT (Debian/Ubuntu)

# Repository Management
sudo add-apt-repository ppa:user/repository-name
sudo apt update
sudo apt upgrade -y

# Package Operations
sudo apt install package-name
sudo apt remove package-name
sudo apt autoremove
sudo apt search keyword

3. User Administration

User Management

# User Operations
sudo useradd -m -s /bin/bash username
sudo usermod -aG sudo username
sudo passwd username
sudo userdel -r username

# Group Operations
sudo groupadd groupname
sudo groupmod -n newname oldname
sudo groupdel groupname

Best Practices

Key points to remember:

  • Always backup before making system changes
  • Use version control for configuration files
  • Document all system modifications
  • Implement monitoring and alerting
  • Regularly update and patch systems
